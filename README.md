# AKQATest

Solution has 4 projects:-
	> AKQA.App.BAL - Contains Business Logic
	> AKQA.App.API - Contains WebAPI
	> AKQA.App.WEB - Has UI
	> AKQA.App.UnitTest - Contains Test Cases
  
  
 Dependencies:
 > .NET Framework 4.7.2
> JQuery
> Bootstrap
> MsTest
> Moq
> Unity


Objective: Capture a persons' name and number, convert number into words
